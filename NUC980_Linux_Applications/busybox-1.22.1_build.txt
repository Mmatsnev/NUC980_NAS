交叉编译：
make menuconfig

1.交叉编译器前缀
Busybox Settings ---->
  Build Options ---->
    () Cross Compiler prefix

输入：arm-nuvoton-linux-uclibceabi-

2.静态编译

Busybox Settings ---->
  Build Options ---->
    [*] Build BusyBox as a static binary (no shared libs)

3.不安装到/usr目录

Busybox Settings --->
  General Configuration --->
    [*] Don't use /usr

4.安装目录

Busybox Settings --->
  Installation Options ("make install" behavior) --->
    BusyBox installation prefix
输入：/home/wcl/nuc980bsp/20190808/NUC980_Linux_Applications/install

make;
make install; //安装到/home/wcl/nuc980bsp/20190808/NUC980_Linux_Applications/install/下
